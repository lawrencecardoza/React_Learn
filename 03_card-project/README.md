# 💼 Job Openings UI (React)

A modern and responsive **Job Openings UI** built using React. This project displays job listings from top companies with clean card-based design.

---

## 🚀 Features

* 📌 Display job openings in card format
* 🏢 Company logo, name, and role
* 🕒 Posted date (e.g., 5 days ago)
* 💼 Job type (Full-time / Part-time)
* 📊 Experience level (Entry / Mid / Senior)
* 💰 Pay per hour
* 📍 Location (including Remote jobs)
* 🔖 Save button (UI only)
* 🎯 Apply button

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS
* Lucide React Icons
